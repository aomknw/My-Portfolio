Table Now - Restaurant booking Mobile App

2024 | UX/UI Design

Overview

Table Now is an application designed to help users discover restaurants and book tables in advance with an easy and secure deposit system. It is ideal for modern users who value convenience in managing their dining plans.

Goals

Improve the ticket booking experience by providing a user-friendly interface and efficient management tools for movie listings and seat reservations.

Problem Statement

The previous booking systems were outdated and cumbersome, leading to poor user satisfaction and operational inefficiencies.

User Persona

Our primary user is a tech-savvy individual who values quick and seamless online booking experiences.

User Flow

The system guides the user from selecting a movie and showtime to reserving seats and completing payment.

Key Features

Interactive movie selection, real-time seat availability, secure payment processing, and detailed booking confirmation.

Typography & Color Palette

Colors: Blue (#0f52ba), White (#ffffff), Gray (#f7f7f7). Typography: Arial, sans-serif.
